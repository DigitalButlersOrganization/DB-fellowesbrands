# DB-Fellowesbrands

> Node >= 16.0.0

## #1 Install
  make `make install` <br/> 
  npm `npm install` <br/>
  yarn `yarn install`

## #2 Start
  make `make serve` <br/>
  npm `npm serve` <br/>
  yarn `yarn serve`

## #3 Format (prettier + autoprefixer) 
make `make format` <br/>
npm `make format` <br/>
yarn `yarn format`

### Troubles
> BrowserSync

`npm install -g browser-sync` or `yarn add -g browser-sync`
