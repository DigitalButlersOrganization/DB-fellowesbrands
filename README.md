# DB-Fellowesbrands

> Node >= 16.0.0

## #1 Install
  Make `make install` <br/> 
  Npm `npm install` <br/>
  Yarn `yarn install`

## #2 Start
  Make `make serve` <br/>
  Npm `npm serve` <br/>
  Yarn `yarn serve`

## #3 Format (prettier + autoprefixer) 
Make `make format` <br/>
Npm `make format` <br/>
Yarn `yarn format`

### Troubles
> BrowserSync

`npm install -g browser-sync` or `yarn add -g browser-sync`
