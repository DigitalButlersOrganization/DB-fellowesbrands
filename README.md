# DB-Fellowesbrands

> Node >= 16.0.0

## #1 Install
`make install` | `npm install` | `yarn install`

## #2 Start
`make serve` | `npm serve` | `yarn serve`


## #3 Format (prettier + autoprefixer) 
`make format` | `make format` | `yarn format`

### Troubles
> BrowserSync

`npm install -g browser-sync` or `yarn add -g browser-sync`
