# DB-Fellowesbrands

> Node >= 16.0.0

## #1 Install
`make install` <br/> 
`npm install` <br/>
`yarn install`

## #2 Start
`make serve` <br/>
`npm serve` <br/>
`yarn serve`


## #3 Format (prettier + autoprefixer) 
`make format` <br/>
`make format` <br/>
`yarn format`

### Troubles

`npm install -g browser-sync` or `yarn add -g browser-sync`
